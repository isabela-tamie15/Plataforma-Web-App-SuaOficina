# Plataforma Web App SuaOficina

Este é o repositório principal da **Plataforma Web App SuaOficina**, um sistema completo desenvolvido para a gestão e modernização de oficinas mecânicas.

## 🚀 Estrutura do Projeto

O projeto é dividido em dois módulos principais:
- **[Frontend](./frontend/)**: Interface do usuário desenvolvida em React com Vite.
- **[Backend](./backend/)**: API responsável pelas regras de negócio e acesso ao banco de dados.

## 📄 Documentação

Toda a documentação técnica, de produto e arquitetural está localizada na pasta `docs/` e arquivos na raiz:
- **`spec.md`**: Especificação base do produto.
- **`context_ai.md`**: Contexto do projeto formatado para LLMs.
- **`docs/`**: Contratos de API, diagramas de arquitetura, roadmap, entre outros.

## 🛠️ Como Executar o Projeto Localmente

### Backend
1. Entre na pasta do backend: `cd backend`
2. Instale as dependências: `npm install`
3. Inicie o servidor em modo de desenvolvimento: `npm run dev`

### Frontend
1. Entre na pasta do frontend: `cd frontend`
2. Instale as dependências: `npm install`
3. Inicie a aplicação web: `npm run dev`
